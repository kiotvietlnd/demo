1. Thiết lập kết nối

Thiết lập → Quản lý chi nhánh → Điền tài khoản kết nối.

2. Thêm mới Thuốc

Hàng hóa → Danh sách hàng hóa → Tạo mới → Thuốc.

3. Nhập hàng theo Lô - HSD

Chọn "Thêm mới lô, hạn sử dụng" khi tạo phiếu nhập.

4. Bán hàng theo Lô

Hệ thống tự động gợi ý Lô hết hạn trước (FEFO).
